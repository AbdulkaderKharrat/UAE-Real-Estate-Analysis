\# UAE Real Estate Analysis



\## Project Overview



This project explores real estate market data in the United Arab Emirates (UAE) to identify pricing trends, property characteristics, and factors affecting property values.



The analysis focuses on understanding how property size, location, furnishing status, bedrooms, and bathrooms influence market prices.



\---



\## Dataset



The dataset contains UAE real estate listings including:



\- Property price

\- Property size

\- Furnishing status

\- Number of bedrooms

\- Number of bathrooms

\- Property location

\---



\## Key Findings



\- Property size shows a positive relationship with price.

\- Certain locations have significantly higher average property prices.

\- Furnished properties tend to be priced differently than unfurnished properties.

\- Bedrooms and bathrooms contribute to overall property valuation.

\- Premium districts dominate the highest-priced listings.
